# lektor-serve
Inspired by `BretFisher/jekyll-serve` but for [Lektor](https://github.com/lektor/lektor), a static site generator written in Python.

Aim: Intended for editing an existing lektor project for a site across different systems without changing the system installation. May be useful for collaborating with others. Please use a production server to server your static site to the world. (Not for production but for preview and collaboration.)
