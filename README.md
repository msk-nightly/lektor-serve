# lektor-serve
Inspired by `BretFisher/jekyll-serve` but for [Lektor](https://github.com/lektor/lektor), a static site generator written in Python.

Aim: Intended for editing an existing lektor project for a site across different systems conveniently (without changing the system installation or creating a new virtual environment and so on). May be especially useful for collaborating with others. Please use a production server to server your static site to the world. (Not for production but for development, preview and collaboration.)
